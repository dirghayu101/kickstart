To start psql: brew services start postgresql

To restart psql: brew services restart postgresql@14

To stop psql: brew services stop postgresql

Create database in psql: createdb <dbName>

Delete database in psql: dropdb <dbName>

Opening the psql terminal for a database: psql <databaseName>

To list all the databases in psql: \list or \l

To quit psql: \q

To get the list of all the schemas/table present in psql of a database: \d

To display the connection information in the psql interface: \conninfo

To display all the users in the database in the psql interface: \du

To drop a table in psql: drop table <tableName>;
